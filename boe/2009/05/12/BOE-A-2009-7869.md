# Resolución de 31 de marzo de 2009, del Ayuntamiento de Culleredo (A Coruña), referente a la convocatoria para proveer varias plazas.

En el Boletín Oficial de la provincia se publican, íntegramente, las bases generales (número 293, de 22 de diciembre de 2008), y específicas (número 72, de 30 de marzo de 2009) que regirán la convocatoria para la provisión por el sistema de concurso-oposición libre de las siguientes plazas de personal laboral fijo:

|  | Puesto | Número 
de plazas |  |
| --- | --- | --- | --- |
|  | Profesor piano | 4 |  |
|  | Profesor piano/acordeón | 1 |  |
|  | Profesor guitarra | 2 |  |
|  | Profesor clarinete | 1 |  |
|  | Profesor trompeta | 1 |  |
|  | Profesor flauta | 1 |  |
|  | Profesor canto | 1 |  |
|  | Profesor composición | 1 |  |
|  | Profesor idiomas | 1 |  |
|  | Profesor cello | 1 |  |
|  | Profesor violín | 2 |  |
|  | Profesor saxofón | 1 |  |

El plazo de presentación de solicitudes será de veinte días naturales, a contar del siguiente al de la publicación del presente anuncio en el BOE.

Los sucesivos anuncios relacionados con esta convocatoria se publicarán, únicamente, en el BOP, en el tablón de anuncios del Ayuntamiento.

Culleredo, 31 de marzo de 2009.–La Concejala de Economía y Hacienda, Régimen Interior y Seguridad, Alicia Ferreiro Varela.
