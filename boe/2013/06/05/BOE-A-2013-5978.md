# Resolución de 3 de junio de 2013, de la Universidad Internacional Menéndez Pelayo, por la que se amplía el plazo de solicitud de subvenciones para la asistencia a las actividades académicas que se celebrarán en la Universidad Internacional Menéndez Pelayo durante el curso 2013, en la sede de Santander.

Se habilita un plazo especial para la solicitud de becas hasta el 7 de junio de 2013, para los cursos siguientes:

| 61ZU | Taller Cadena COPE. |
| --- | --- |
| 61ZS | La autoridad fiscal independiente: un reto actual, pasado y futuro. |
| 61VD | Semana de Proyectos Europeos Flagship: Grafeno y Cerebro Humano. |
| 61ZT | Proyectos Europeos Flagship: Grafeno. |
| 61ZY | Proyectos Europeos Flagship: Cerebro Humano. |
| 61XM | XII Bienal de Arquitectura Española. |
| 61TZ | Empleos de futuro. |
| 61XF | Posicionamiento de los medicamentos. |
| 61QX | Vinos andaluces: compartiendo sabor, generando cultura. |
| 61Y5 | Éxito del deporte. |
| 61QP | Taller de radio. La programación radiofónica: presente y futuro. 
 Onda cero un modelo. |
| 61RU | Las excavaciones arqueológicas en el templo de Millones de años de Tutmosis III; Luxor, Egipto. |
| 61VH | La instrucción penal: ¿juez inquisitivo o juez de garantías? El papel del fiscal. |

Cursos que se impartan del 29 de julio al 13 de septiembre, el plazo se ampliará hasta el 21 de junio de 2013.

Madrid, 3 de junio de 2013.–El Rector de la Universidad Internacional Menéndez Pelayo, César Nombela Cano.
