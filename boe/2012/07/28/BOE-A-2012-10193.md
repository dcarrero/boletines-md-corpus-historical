| Conocimientos de Propiedad Industrial. | 2,00 |  |  |
| Experiencia en utilización de programas informáticos a nivel de usuario y manejo de Bases de Datos para grabación y consulta, así como en la elaboración de estadísticas. | 2,00 |  |  |
