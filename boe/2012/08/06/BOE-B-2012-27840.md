# Anuncio del Instituto de Parques y Jardines de Barcelona de rectificación del pliego de cláusulas particulares y anuncio de licitación del contrato de servicios de mantenimiento de infraestructuras en los parques y jardines de la ciudad para los controles del Área I-X (Distritos de Ciutat Vella y Sant Martí), Área IV-V (Districtes de Les Corts i Sarrià - Sant Gervasi) i el Área VI-VII (Districtes Gràcia i Horta-Guinardó).

El Gerente del Instituto de Parques y Jardines de Barcelona, en fecha 30 de julio del 2012, ha resuelto:

Rectificar el pliego de cláusulas particulares del contrato de servicios de mantenimiento de infraestructuras en los parques y jardines de la ciudad para los controles del Área I-X (Distritos de Ciutat Vella y Sant Martí), Área IV-V (Districtes de Les Corts i Sarrià - Sant Gervasi) i el Área VI-VII (Districtes Gràcia i Horta-Guinardó), de manera que, en su anexo 5, donde dice:

| 2 | MATERIALES |  |  |
| --- | --- | --- | --- |
|  | Concepto | Superfície (m2) |  |
|  | Área I - X | 1.955.380 | 40.984,76 |
|  | Área IV - V | 1.542.246 | 32.325,48 |
|  | Área VI - VII | 1.320.232 | 27.672,06 |
|  | TOTAL MATERIALES | 100.982,30 |  |

Ha de decir:

| 2 | MATERIALES |  |  |
| --- | --- | --- | --- |
|  | Concepto | Superfície (m2) |  |
|  | Área I - X | 1.955.380 | 34.732,85 |
|  | Área IV - V | 1.542.246 | 27.394,47 |
|  | Área VI - VII | 1.320.232 | 23.450,90 |
|  | TOTAL MATERIALS | 85.578,22 |  |

Dado que los importes incluidos en el mismo eran con el IVA incluido.

Ampliar la fecha límite de la presentación de las proposiciones hasta el dia 14 de septiembre a las 14:00 horas; y Aplazar las fechas de apertura del sobre 2A hasta el día 25 de septiembre a las 14:00 horas i el sobre 2-B hasta el dia 2 de octubre, a las 14:00 horas.

El anuncio de licitación se publicó en el Boletín Oficial del Estado número 174, de fecha 21 de julio de 2012.

Barcelona, 31 de julio de 2012.- Iolanda Garcia i Ceprià, P.D. de 27 de octubre de 2009, Secretaria Delegada Accidental.
