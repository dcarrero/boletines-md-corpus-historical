 Ejecución de programas de homologación.

 Realización, supervisión y verificación de estudios y análisis técnicos. |
| Productos anticorrosivos y de protección. |  |  |  |
| Productos especiales y de usos diversos. |  |  |  |
| Aceites lubricantes. | Laboratorio de Lubricantes. |  |  |
| Grasas. |  |  |  |
| Líquidos hidráulicos. |  |  |  |
