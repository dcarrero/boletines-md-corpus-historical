# Corrección de errores de la Resolución de 2 de noviembre de 2017, de la Presidencia de la Agencia Estatal de Investigación, por la que se convoca la provisión de puestos de trabajo por el sistema de libre designación.

Publicada en el «Boletín Oficial del Estado» n.º 277, de 15 de noviembre de 2017, la Resolución de 2 de noviembre de 2017, de la Presidencia de la Agencia Estatal de Investigación, por la que se convoca la provisión de puestos de trabajo por el sistema de libre designación, y advertido error material consistente en haber incluido en la convocatoria dos puestos de Secretario / Secretaria de Director General, y comprobado que uno de los mismos se encontraba en situación de reserva por lo que no podría ser ofertado, procede la rectificación del Anexo I de la Resolución de 2 de noviembre de 2017, en los siguientes términos:

ANEXO I

Corrección de errores:

Página 109765, donde dice:

| «N.º Orden | Centro directivo puesto de trabajo | N.º Puesto | Localidad | Subgrupo/Admon. | Nivel | Compl. Específico |
| --- | --- | --- | --- | --- | --- | --- |
|  | Agencia Estatal de Investigación. |  |  |  |  |  |
| Unidad de Apoyo. |  |  |  |  |  |  |
| 2-3 | Secretario/Secretaria de Director General. | 2 | Madrid. | C1-C2/AE | 17 | 7.202,30 €» |

Debe decir:

| «N.º Orden | Centro directivo puesto de trabajo | N.º Puesto | Localidad | Subgrupo/Admon. | Nivel | Compl. Específico |
| --- | --- | --- | --- | --- | --- | --- |
|  | Agencia Estatal de Investigación. |  |  |  |  |  |
| Unidad de Apoyo. |  |  |  |  |  |  |
| 2 | Secretario/Secretaria de Director General. | 1 | Madrid. | C1-C2/AE | 17 | 7.202,30 €» |
