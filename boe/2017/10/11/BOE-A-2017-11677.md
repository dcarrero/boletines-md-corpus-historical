# Corrección de errores de la Resolución de 28 de agosto de 2017, de Parques Nacionales, por la que se publica el Convenio con el Consell Insular de Menorca por el que se encomienda al Consell la gestión de la finca Alforí de Dalt.

Advertido error en la Resolución de 28 de agosto de 2017, de Parques Nacionales, por la que se publica el Convenio con el Consell Insular de Menorca por el que se encomienda al Consell la gestión de la finca Alforí de Dalt, publicada en el «Boletín Oficial del Estado» número 216, de 8 de septiembre, se procede a subsanarlo mediante la oportuna rectificación:

En la página 89054, donde dice:

«Consell Insular de Menorca

| Anualidad | Total |
| --- | --- |
| 2017 | 10.000 |
| 2018 | 10.000 |
| 2019 | 10.000 |
| 2020 | 10.000 |
| 2011 | 5.000» |

Debe decir:

«Consell Insular de Menorca

| Anualidad | Total |
| --- | --- |
| 2017 | 10.000 |
| 2018 | 10.000 |
| 2019 | 10.000 |
| 2020 | 10.000 |
| 2021 | 5.000» |
