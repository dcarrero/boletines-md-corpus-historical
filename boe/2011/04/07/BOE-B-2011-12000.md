# Resolución de 23 de marzo de 2011 de la Subsecretaría por la que se hace pública la corrección de errores en pliego de prescripciones técnicas y nueva fechas de presentación y apertura de proposiciones del expediente 103/11 Suministro y actualización de productos Microsoft con destino al Ministerio de la Presidencia.

Rectificación del pliego de prescripciones técnicas y nuevos plazos:

1. En la página 4, en el apartado Entorno licencias servidor, en el punto 2 se decía: "Renovación de cinco (4) unidades del sistema operativo para servidores "Windows Server Enterprise". Debe decir: "Renovación de cuatro (4) unidades del sistema operativo para servidores "Windows Server Enterprise"

2. En la página 5, apartado CUADRO GENERAL DE LICENCIAS, se decía: "Windows Server Standard………4. Windows Server Enterprise…….10. Debe decir:" Windows Server Standard………..10. Windows Server Enterprise………4"

3. Fecha límite presentación de ofertas hasta las 14:00 horas del 3 de mayo de 2011.

4. Apertura de ofertas el 19 de mayo de 2011 a las a 10:00 horas.

El anuncio de licitación se publicó en el BOE número 59, de 10 de marzo de 2011, página 25468

Madrid, 29 de marzo de 2011.- La Subsecretaria.
